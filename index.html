Configuration Automation

Lately, I've been working on automating the setup of my preferred working environment. It is a rather simple project, but, then again, that is usually a good thing. Anyways, the only novel part (so far) is the customization script that allows you to customize configuration files when they are installed. This is useful for configs that require a password (e.g. irssi).

GeoJSON ToolKit

While working on my Senior Design project, I found myself wanting some simple tools for working with GeoJSON data So, I wrote a validator and object counter and put them in a public repository. There are currently Python and JavaScript implementations, and they are available on PyPI and npm, respectively.

Senior Design Project

In response to a proposal made by LLNL for a system that could fuse data and interact with networked sensors, our team built a mobile application and a web application to interface a custom pluggable sensor and a remote database. Gathered data can be merged with publicly available information (e.g. Twitter) to allow users to compare and relate large amounts of data.

As Fusion/Visualization Lead and Treasurer, I contributed code and graphics while managing our budget (approximately $2500). More specifically, I built OGRe, a library for fetching data from 3rd-party APIs, and Vizit, a visualizer that can render any GeoJSON file.

Sentinel

While trying to devise a better way of managing passwords for many digital accounts, I came up with a memorable scheme that I implemented in an encrypted spreadsheet file. Thinking it would help more people if it were a standalone application, I created a web2py project for a class at UCSC.

Update: I will not be contributing to this project any further. Turns out, I just wanted Keepass all along...
