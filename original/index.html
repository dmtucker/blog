GeoJSON ToolKit

While working on my Senior Design project, I found myself wanting some simple tools for working with GeoJSON data So, I wrote a validator and object counter and put them in a public repository. There are currently Python and JavaScript implementations, and they should be available on PyPI and npm, respectively.

Senior Design Project

In response to a proposal made by LLNL for a system that could fuse data and interact with networked sensors, our team built a mobile application and a web application to interface a custom pluggable sensor and a remote database. Gathered data can be merged with publicly available information (e.g. Twitter) to allow users to compare and relate large amounts of data. More details are available in our poster and even more in our report.

As Fusion/Visualization Lead and Treasurer, I contributed code and graphics while managing our budget (approximately $2500). More specifically, I built OGRe, a library for fetching data from 3rd-party APIs, and Vizit, a visualizer that can render any GeoJSON file.
